### 3. Mages as Self-Calibrating Weights

**The inversion that makes magic unique:**

In ANNs:
- Nodes are fixed (same activation function)
- Weights are learned (adjusted by gradient descent)
- External training signal determines adjustment

In magic:
- Mages ARE adaptive—they calibrate themselves
- Each Mage adjusts their own "weights" through practice
- No external training signal—**resonance** provides the gradient

**Implication:** Mages are both nodes AND self-adjusting weights. Each node is responsible for its own calibration. This is:
- Decentralized learning
- Autonomous adaptation
- Sovereignty-preserving network intelligence

**The gradient analog:**

| ML Concept | Magic Analog |
|------------|--------------|
| **Loss function** | Misalignment with reality's fundamental pattern |
| **Gradient** | Resonance feedback (high = aligned, low = misaligned) |
| **Backpropagation** | Reflection on practice outcomes |
| **Weight update** | Mage self-calibration |
| **Learning rate** | How quickly Mage integrates feedback |

**Resonance IS the gradient signal.** It tells each Mage how to adjust without external authority.

## What Makes Magic Different from Standard Networks

| Standard Neural Network | Magic Network |

| Weights are global | "Weights" are local (each Mage calibrates self) |
