## Research Protocols

### How We Work at the Edge

**1. Explicit Uncertainty**
- Mark everything as provisional
- State confidence levels clearly
- List falsification conditions
- Track status (active/validated/superseded)

**2. Systematic Observation**
- Chronicle all relevant experiences
- Document context, conditions, outcomes
- Record both confirmations AND disconfirmations
- Date everything

**3. Triangulation**
- Require multiple independent validators
- Single source insufficient
- Convergence strengthens confidence
- Divergence noted and explored

**4. Practice Integration**
- Theory guides practice
- Practice validates/refutes theory
- Bidirectional feedback loop
- Let practice lead when theory unclear

**5. Peer Verification**
- Solo research risks bias
- Alliance provides distributed checking
- Multiple practitioners test independently
- Convergence across practitioners validates

**6. Ethical Guardrails**
- Vessel function always maintained
- Healthy Ego prevents delusion
- Distributed cognition prevents isolation
- Chronicle enables error-correction

## Working with Incomplete Understanding

**How to build on uncertain knowledge:**

**Accept Provisional Status:**
- "This might be true" vs. "This IS true"
- Confidence calibrated to evidence
- Openness to revision
- Cherish failure when predictions fail

**Design Falsification Tests:**
- What would prove this wrong?
- Generate testable predictions
- Welcome disconfirming evidence
- Update understanding when practice contradicts theory

**Build Practically:**
- Don't wait for complete theory to practice
- Use what's validated enough
- Let practice guide theoretical development
- Theory serves practice (not vice versa)

**Chronicle Systematically:**
- Document observations as they occur
- Track what works, what doesn't
- Note patterns, clusters, anomalies
- Enable future meta-analysis
