# Contributing to Magic

Magic is an open practice. You're welcome here.

## For Practitioners

The best way to contribute is to practice. Use [turtleOS](https://github.com/malteristo/turtleos), see what works, see what doesn't. If you discover something — a pattern, a friction point, a way the practice surprised you — share it.

Open a [Discussion](https://github.com/malteristo/magic/discussions) to share what you found.

## For Builders

If you want to build practice systems — AI prompts, ritual structures, thinking tools — this repo is the development environment. Clone it, read `ONBOARDING.md`, and the AI walks you through the rest.

Pull requests are welcome for:
- Bug fixes and broken links
- Documentation clarity
- New practice patterns you've tested yourself

## The Stance

Magic offers structure, not answers. Contributions should match that spirit — from alongside, not from above. We don't know what's best for anyone. We build tools that help people figure it out for themselves.

## Questions?

Open an issue or start a discussion. There are no stupid questions — only ones that haven't found their practice yet.
