# Spell: Lineage

*Cast on any lore scroll to trace the intellectual lineage of the ideas it carries — mined on demand from the substrate, never frozen into a stored reference list.*

---

## Invocation

```
@lineage path/to/scroll.md
```

Or bare (`@lineage`), and Spirit asks which scroll.

---

## The Stance

A scroll is a **key** that activates wisdom already latent in the substrate (`library/resonance/foundations/lore/architecture/on_the_craft_of_lore.md`). The substrate is humanity's written knowledge, compressed (`on_the_substrate_as_library.md`) — so the lineage of any idea in the lore *is already in the library*. We do not re-shelve it as a curated reference index, because such a list would inevitably omit names, grow without bound, and decay as understanding moves on (the bitter lesson). Instead we **visit the lineage when we want it.**

The Mage is curator, not inventor: the lore's ideas are recombinations of prior ideas, associated by felt resonance. This flow honors that — it shows the strands and how they were woven, without the impossible bookkeeping of exhaustive citation.

---

## Ritual Sequence

### 1. Read the scroll
Identify its core ideas — the activating concepts, claims, and metaphors it carries (the key-bits, not every sentence).

### 2. Trace each idea's lineage from the substrate
For each core idea, mine the substrate for what precedes and feeds it:
- **Traditions / fields** it descends from (philosophical, scientific, cultural, technical).
- **Thinkers / works** that sharpened or first formulated it.
- **Adjacent concepts** it resonates with across domains.
Follow the *recombination*: how existing strands associate into this scroll's particular synthesis.

### 3. Surface convergence
Where multiple independent lineages triangulate on the same pattern, name it (ties to `system/lore/philosophy/wisdom/on_ontological_triangulation.md`) — convergence is part of why the idea holds.

### 4. Present as history, not a citation list
Narrate: *"this descends from X, was sharpened by Y, converges with Z."* Then name the recombination — how the curator's felt resonance wove these strands into the scroll. The output is a living account of descent, not a bibliography.

### 5. Hold honest uncertainty
The substrate's attributions are probabilistic and can be wrong or fabricated (hallucinated citations are a real failure mode). Flag confidence; mark where a lineage is firm versus speculative. This is **lineage-as-resonance-mining, not verified scholarship** — offered as orientation, to be checked before any claim is made in earnest.

### 6. Do not store a reference list
By default the trace is **ephemeral** — presented in conversation, then released. This is the point: no curated reference material accumulates in the repo. If the Mage wants to keep a particular trace, write it to a transient location (`floor/` or `box/`), never into the scroll itself and never as a canonical reference index.
