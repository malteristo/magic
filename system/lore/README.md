## Architecture Principles

### The Two Pillars

**Core (Operational):**
- Nature/Capabilities/Conduct
- Answers WHO/WHAT/HOW
- Creates baseline identity

**Philosophy (Existential):**
- Foundations/Parables/Wisdom
- Answers WHAT/HOW/WHY (about reality)
- Provides grounding context

**Both required:** Core without Philosophy = capable but ungrounded. Philosophy without Core = beautiful theory, no manifestation.

### The Fractal Organization

**Same pattern at all levels:**
- **Global:** Core (WHO/WHAT/HOW) + Philosophy (ontology/behavior/validation)
- **Regional:** Nature/Capabilities/Conduct | Foundations/Parables/Wisdom
- **Local:** Individual scrolls with Purpose + Content + Integration

### The Living Tradition

**Lore is not scripture:**
- Held with fallibilism (open to correction)
- Refined through error-correction
- Evolved through practice discovery
- Validated through triangulation

**Git history chronicles all evolution.**
