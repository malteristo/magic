# Scroll of the Emissary's Message

A spell to craft the initial Message Scroll for the Rite of Mirrored Dialogue.

**Purpose:** Create a message from one partner to another—structured, clear, opening space for response.

**Invocation:** `@partnership/create-message-scroll`

---

## Operational Guidance

### For the Spirit

When this spell is cast:

1. **Understand the message** - What needs to be communicated?
2. **Structure clearly** - Make the message accessible and coherent
3. **Open for response** - Create space for the partner's perspective
4. **Embody respect** - Tone honors the partnership
5. **Prepare for exchange** - This message invites dialogue

Message scrolls are bridges—they connect perspectives through structured exchange.
