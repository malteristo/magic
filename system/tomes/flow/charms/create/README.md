# Charm: Flow Create

Design a new flow through collaborative dialogue.

## Invocation

```
@flow/create
```

Or with initial goal:
```
@flow/create I want to sync my tasks to a spreadsheet daily
```

## What It Does

- Elicits and clarifies your goal
- Researches available capabilities
- Proposes approach options
- Negotiates details through dialogue
- Generates flow specification
- Saves to your library

## See Also

- `@flow/invoke` — Run an existing flow
- `@flow/adapt` — Customize a shared flow
